# 1.1 Security Controls

## Goal

Compare and contrast various types of security controls.

## Security Control Categories

- **Technical:** controls enforced by systems or software
- **Managerial:** policies, risk management, governance decisions
- **Operational:** day-to-day procedures run by people
- **Physical:** barriers and protections for facilities and hardware

## Security Control Types

- **Preventive:** stop an incident before it happens
- **Deterrent:** discourage attackers from attempting an action
- **Detective:** identify events during or after they occur
- **Corrective:** fix systems after an incident
- **Compensating:** alternate control when the ideal one is unavailable
- **Directive:** guide behavior through required rules and standards

## Quick Review

- Categories describe **where** the control sits.
- Types describe **what** the control is meant to do.
