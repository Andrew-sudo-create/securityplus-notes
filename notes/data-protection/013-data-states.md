Data States - The condition of data at a given time and how it should be protected.

Data at Rest - Data stored on disks, databases, backups, or cloud storage.

Data in Transit - Data moving across a network between systems.

Data in Use - Data currently being processed in memory by applications or users.

Security focus:
- At Rest: encryption, access control, key management
- In Transit: TLS, VPNs, secure protocols
- In Use: least privilege, memory protections, endpoint security
