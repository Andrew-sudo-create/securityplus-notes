Data Classification - Category based on the organizations value and the sensitivity of the info
if it were to be disclosed.

Sensitive Data - Any information that can result in a loss of security or a loss of advantage to a company if accessed by an unauthorized person.

We need to be protecting sensitive data.

Thats why we categorise data

Overclassifying data leads to protecting data at a high level.

2 Different classification schemes: Commercial Business, Government

Commercial Business - Uses 1 of 4 common classification levels, Public, Sensitive, Private, Confidential

Public Data - No impact if its released.
Sensitive - Has minimal impact if released.
Private - Contains data that should only be used by organisation ( Private data is anyinformation that relates to an indicidual entity)
Confidential - Contains items such as trade secrets, IP, data and source code that affect business if disclosed.
Critcal - Contains valueble data.

Government Classifications - Unclassified, Sensistive but unclassified, Confidential, Secret, Top secret.

Unclassified - Genrerally can be released.
Sensitive but unclassified - Data that wont hurt national securtiy but will hurt those whose data is there.
Confidential - Data that could seriously affect government secrets.
Secret - Data that could seriously damage national security if disclosed.
Top Secret - Data that would gravely damage national security if disclosed..
